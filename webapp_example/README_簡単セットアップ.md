# 他のパソコンから使えるようにする方法 - 簡単セットアップ

## 🎯 目的

ノートパソコンを持っていない他のパソコンからも、ブラウザで価格自動取得システムを使えるようにします。

## 📋 方法の選択

### 方法1: ローカルネットワークで公開（最も簡単・推奨）⭐

**メリット**:
- ✅ セットアップが簡単（5分で完了）
- ✅ 無料
- ✅ セキュア（インターネットからはアクセス不可）
- ✅ 既存のコードをそのまま使用可能

**デメリット**:
- ⚠️ ノートパソコンを起動している必要がある
- ⚠️ 同じWi-Fi/ネットワークに接続している必要がある

**手順**:
1. `webapp_example`フォルダに移動
2. `pip install -r requirements_web.txt` で依存パッケージをインストール
3. `python run_local_network.py` で起動
4. 表示されたURLを他のパソコンのブラウザで開く

詳細は `使い方_ローカルネットワーク.md` を参照してください。

---

### 方法2: クラウドにデプロイ（本格的）

**メリット**:
- ✅ どこからでもアクセス可能
- ✅ ノートパソコンを起動している必要がない
- ✅ 24時間利用可能

**デメリット**:
- ⚠️ セットアップが複雑
- ⚠️ 費用がかかる場合がある（無料枠あり）

**推奨サービス**:
- **Heroku**（簡単・無料枠あり）
- **AWS**（本格的）
- **Google Cloud Platform**（本格的）

詳細は `ウェブアプリ化_デプロイ計画.md` を参照してください。

---

## 🚀 すぐに始める（方法1: ローカルネットワーク）

### ステップ1: 依存パッケージのインストール

```bash
cd webapp_example
pip install -r requirements_web.txt
```

### ステップ2: ウェブアプリの起動

```bash
python run_local_network.py
```

起動すると、以下のようなメッセージが表示されます：

```
============================================================
価格自動取得システム - ウェブアプリケーション
============================================================

ローカルネットワークでアクセス可能にしました！

以下のURLからアクセスできます：
  このパソコンから: http://localhost:5000
  他のパソコンから: http://192.168.1.100:5000

⚠️  注意:
  - 同じWi-Fi/ネットワークに接続している必要があります
  - ファイアウォールの設定が必要な場合があります

サーバーを停止するには Ctrl+C を押してください
============================================================
```

### ステップ3: アクセス

#### このパソコンから
ブラウザで `http://localhost:5000` にアクセス

#### 他のパソコンから
ブラウザで `http://[表示されたIPアドレス]:5000` にアクセス
（例: `http://192.168.1.100:5000`）

### ステップ4: 使い方

1. ブラウザでアクセス
2. 「実装済み18社の価格を取得」ボタンをクリック
3. スクレイピングが完了するまで待機（約1-2分）
4. 結果が表示されます
5. 「Excelダウンロード」ボタンでExcelファイルをダウンロード

## 🔧 トラブルシューティング

### 接続できない場合

1. **同じネットワークに接続しているか確認**
   - 同じWi-Fiに接続している必要があります

2. **ファイアウォールの設定**
   - macOS: システム環境設定 → セキュリティとプライバシー → ファイアウォール
   - Windows: コントロールパネル → Windows Defender ファイアウォール

3. **IPアドレスを確認**
   - `run_local_network.py` を再起動してIPアドレスを確認

詳細は `使い方_ローカルネットワーク.md` を参照してください。

## 📝 注意事項

- **ノートパソコンを起動している必要があります**
  - ノートパソコンを閉じたり、スリープさせると接続できなくなります
  - 常時起動が必要な場合は、クラウドデプロイを検討してください

- **セキュリティ**
  - ローカルネットワーク内のみアクセス可能です
  - インターネットからはアクセスできません（安全）
