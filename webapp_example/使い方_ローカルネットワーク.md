# ローカルネットワークで他のパソコンから使う方法

## 📋 概要

この方法を使えば、**同じWi-Fi/ネットワークに接続している他のパソコンからも、ブラウザで価格自動取得システムを使えます**。

## 🚀 セットアップ手順

### 1. 依存パッケージのインストール

```bash
cd webapp_example
pip install -r requirements_web.txt
```

### 2. ウェブアプリの起動

```bash
python run_local_network.py
```

起動すると、以下のようなメッセージが表示されます：

```
============================================================
価格自動取得システム - ウェブアプリケーション
============================================================

ローカルネットワークでアクセス可能にしました！

以下のURLからアクセスできます：
  このパソコンから: http://localhost:5000
  他のパソコンから: http://192.168.1.100:5000

⚠️  注意:
  - 同じWi-Fi/ネットワークに接続している必要があります
  - ファイアウォールの設定が必要な場合があります

サーバーを停止するには Ctrl+C を押してください
============================================================
```

### 3. アクセス方法

#### このパソコンから
ブラウザで `http://localhost:5000` にアクセス

#### 他のパソコンから
ブラウザで `http://[表示されたIPアドレス]:5000` にアクセス
（例: `http://192.168.1.100:5000`）

## 🔧 トラブルシューティング

### ファイアウォールの設定

#### macOS の場合
1. システム環境設定 → セキュリティとプライバシー → ファイアウォール
2. 「ファイアウォールをオンにする」
3. 「オプション...」をクリック
4. Python を許可リストに追加

または、ターミナルで：
```bash
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /usr/local/bin/python3
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --unblockapp /usr/local/bin/python3
```

#### Windows の場合
1. コントロールパネル → システムとセキュリティ → Windows Defender ファイアウォール
2. 「アプリをファイアウォールを通じて許可する」
3. Python を許可リストに追加

### 接続できない場合

1. **同じネットワークに接続しているか確認**
   - 同じWi-Fiに接続している必要があります
   - 有線LANとWi-Fiが別ネットワークの場合は接続できません

2. **IPアドレスを確認**
   - ノートパソコンのIPアドレスが変更されていないか確認
   - `run_local_network.py` を再起動してIPアドレスを確認

3. **ポート番号を確認**
   - デフォルトは5000番ポートです
   - 他のアプリが5000番を使っている場合は変更が必要です

## 📱 使い方

1. ブラウザでアクセス
2. 「実装済み18社の価格を取得」ボタンをクリック
3. スクレイピングが完了するまで待機（約1-2分）
4. 結果が表示されます
5. 「Excelダウンロード」ボタンでExcelファイルをダウンロード

## ⚠️ 注意事項

- **ノートパソコンを起動している必要があります**
  - ノートパソコンを閉じたり、スリープさせると接続できなくなります
  - 常時起動が必要な場合は、クラウドデプロイを検討してください

- **セキュリティ**
  - ローカルネットワーク内のみアクセス可能です
  - インターネットからはアクセスできません（安全）

- **パフォーマンス**
  - 複数の人が同時にスクレイピングを実行すると時間がかかります
  - 1回の実行で約1-2分かかります
